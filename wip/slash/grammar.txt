Slash-0 grammar

Slash0:
  - List of nodes
  - List of commands

node:
  type=node type
node types:
  env: environment variable
  file: file name
        is_directory: true/false
  cell: cell name
        origin: input, intern, extern
        is_array: true/false
        array_length
  variable: (dynamic) variable name
        is_list: true/false
  context: context_name
        is_json: true/false
"file" and "env" are implicitly constructed from code.
The other ones are explicitly declared.

command:
  index: 0 - [number of commands]
  lineno
  source code: str, original source code
  type=:
    "standard"
    "assign", (from =)
        - variable noderef
        - variable expression (see below)
    "cat"
      - list of cell noderefs (is_array must be True)
      - result cell noderef (is_array must be True)
    "read",
      - input cell noderef
      - output variable noderef (is_list=False)
    "lines",
    - input cell noderef
    - output variable noderef (is_list=True)
    "fields",
    - input cell noderef
    - output variable noderef (is_list=True)
    "cell",
    - input variable expression
    - output cell noderef
    "load",
    - input file noderef
    - result cell noderef
    "map"
    - input_dot: True or False
    - input context noderef (only if non-dotted)
    - output file (is_directory=True)

Standard-command grammar:
- command:
    - type: file, or registered command (TODO)
    for type=file:
      - file expression (str with {})
      - noderef list:
        - env
- argument list:
  - type: file, cell or variable expression
    for type=file: see above
    for type=cell:
      - single noderef of type=cell
    for type=variable expression:
       see below
- output list:
  - type: stdout, stderr, stdout+stderr
  - single noderef of type=cell
- capture (>!):
  - single noderef of type=context
     (json must be true)

variable expression:
  is_simple: True if refers to just a variable
  if True: variable noderef
  if False:
  - argument expression (str with {})
  - noderef list:
      - variable

noderef:
    - command index
    - type: env, cell, variable,
        or context
    - index: -1 for NULL node
    - input or output
