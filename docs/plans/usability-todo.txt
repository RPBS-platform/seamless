Bucket list of things that make Seamless easier to work with
as a Mobyle replacement.
Details are/will be documented elsewhere in various plans.

- Automatic generation of web form GUIs
- Logging error messages, cache misses, network traffic, progress bars
- Job/cache control (canceling, job database)
- Better control of execution details (execution options, reactor start/stop side effects, execute Docker transformer as bash transformer)
